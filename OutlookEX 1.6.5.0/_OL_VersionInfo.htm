Function Reference

_OL_VersionInfo

Returns a string holding version/date of OutlookEX_Base, OutlookEX and OutlookEX_GUI.

#Include <OutlookEX_Base.au3>
_OL_VersionInfo()

 

Parameters

None.

 

Return Value

Success: string holding version/date of OutlookEX_Base, OutlookEX and OutlookEX_GUI separated by @CRLF.

 

Remarks

Example of the returned string: "OutlookEX_Base: 1.2.2.0 220-09-01 <@CRLF> OutlookEX: 1.6.3.0 2020-09-01"

 

Related

 

Example


#include <OutlookEX.au3>

;------------------------------------------------------------------------------------------------------------------------------------------------
; Example 1 - Display the version info for this UDF
;------------------------------------------------------------------------------------------------------------------------------------------------
Global $sVersionInfo = _OL_VersionInfo()
MsgBox($MB_ICONINFORMATION, "_OL_VersionInfo", $sVersionInfo)
